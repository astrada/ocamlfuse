# Documentation

This directory stores project documentation for agents and maintainers working
on the OCaml bindings.

- `bindings.md`: architecture and maintenance notes for the camlidl/libfuse
  binding layer.
- `release-notes.md`: release notes for the FUSE 3 package.
- `plans/`: proposed implementation plans for larger changes.
- `plans/archived/`: completed or superseded implementation plans.
- `plans/archived/fuse3/`: completed FUSE 3 migration requirements,
  milestones, and analysis.

The user-facing project overview remains in the top-level `README.md`.
